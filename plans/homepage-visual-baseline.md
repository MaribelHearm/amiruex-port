# 首页视觉基线记录

记录时间：2026-04-09
来源：当前线上截图

## 1. 整体视觉

- 主色调为深蓝/青蓝，背景图为人物插画，整体偏暗。
- 页面采用“前景卡片 + 背景大图 + 网格纹理”结构。
- 信息区块顺序：Hero → Blog → Tools → About/状态卡。

## 2. 顶部区域

- 顶栏居中，导航项为 `Blog / Tools / About / Search`。
- 顶栏外轮廓可见，但灵动岛毛玻璃质感不稳定，模糊感偏弱。
- 页面最顶部存在一条明显深色带，视觉上像遮罩/图层拼接痕迹。

## 3. 背景层观感

- 背景图已显示，但局部存在“横向分层”或“条带感”。
- 背景遮罩与内容层叠加后，右侧区域的透明过渡不够自然。
- 目前效果更接近“暗色透明板”，而不是“高斯磨砂玻璃里的光斑”。

## 4. 卡片与内容层

- 内容卡片边框、圆角、阴影风格统一，基础视觉风格是对的。
- 主卡片可读性尚可，但背景高对比区域会干扰标题阅读。

## 5. 作为重构验收基线的关键对比点

后续重构时，以以下结果作为“已改善”判断标准：

1. 顶部深色横条消失。
2. 灵动岛滚动后出现稳定毛玻璃效果（背景细节被高斯模糊，不是纯黑）。
3. 背景不再出现明显横向拼接/条带感。
4. 卡片与背景分离更清晰，但不丢失沉浸感。
